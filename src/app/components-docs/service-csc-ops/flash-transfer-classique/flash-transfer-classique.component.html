Guide Flash Transfer Classique

Flash Transfer Classique

Pour se connecter, l’operateur saisi le lien ci-dessous

  • https://commons.cfc-rdc.com:3000 , sur la barre d'adresse du navigateur
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Pour passer une opération Flash Transfer Classique, l’opérateur clique sur

  • FLASH TRANSFERT

Une fois sur FLASH TRANSFERT, la page suivante s'affiche

1. ENVOI

Pour faire un envoi, cliquer sur :

  • Envoi
  • Selectionner le pays de destination
  • Suivant

En cliquant sur le bouton Suivant la page suivante s'affiche pour passer les opérations d'envoi

  • Sélectionner « Flash Transfer Classique »

Après avoir cliqué sur « Flash Transfer Classique », la page suivante s’affiche :

Lors de cette opération, l’opérateur :

  • saisit les informations des clients (expéditeur et bénéficiaire) ;
  • renseigne les détails relatifs à la transaction ;
  • scanne la pièce d’identité de l’expéditeur (émetteur) si celui-ci s’enregistre pour la première fois dans le système ;
  • clique sur le bouton « Envoyer » pour confirmer le transfert d’argent, après s’être assuré que toutes les informations saisies sont correctes.

N.B : L’opérateur est appelé à prendre connaissance des indications reprises au préalable dans l'image ci-dessus pour le bon déroulement des opérations

Après s’être rassuré que les éléments introduits sont exacts, l'opérateur peut valider l'opération en cliquant sur le bouton "Oui" Un chargement est effectué afin d’obtenir le reçu de l’opération.

Après avoir obtenu le reçu de l’opération, l’opérateur peut ensuite l’imprimer en cliquant sur le bouton Imprimer

2. RETRAIT

Pour faire un envoi, cliquer sur :

  • Retrait
  • Ensuite, sur l'option classic

En cliquant sur l'onglet Retrait, l’opérateur accède à la page suivante pour effectuer les transactions de paiement.

L’opérateur procède de la manière suivante :

  • Entrer le PCN de l’opération sur le champ
  • Cliquez sur le bouton Rechercher pour lancer la recherche de l’opération.

Une fois l’opération trouvée, la page suivante s’affiche :

  • Renseigner les informations du beneficiaire ;
  • Cliquer sur le bouton Payer

La page suivante s’affiche afin de confirmer le transfert d’argent.

Cliquez sur le bouton « Oui » pour confirmer le transfert d’argent. Le reçu de retrait s’affiche alors, avec la possibilité de l’imprimer en cliquant sur le bouton Imprimer.

3. REPORTING

3.1. ENVOIS

L’opérateur peut consulter les différentes transactions d’envois et de retraits effectués en suivant les étapes suivantes :

  • Cliquez sur « Transaction » ;
  • Cliquez sur « Envois ».

Après avoir cliqué sur « Envois », la page suivante s’affiche avec les options suivantes :

  • Appliquer un filtre en fonction de la période, de la devise ou du statut ;
  • Cliquer sur le bouton « Filtrer » pour voir les différentes transactions ;
  • Exporter le journal des transactions d’envois en cliquant sur le bouton « Exporter ».

3.2. RETRAITS

L’opérateur peut consulter les différentes transactions d’envois et de retraits effectués en suivant les étapes suivantes :

  • Cliquez sur « Transaction » ;
  • Cliquez sur « Retraits ».

Après avoir cliqué sur « Retraits », la page suivante s’affiche avec les options suivantes :

  • Appliquer un filtre en fonction de la période, de la devise ou du statut ;
  • Cliquer sur le bouton « Filtrer » pour voir les différentes transactions ;
  • Exporter le journal des transactions d’envois en cliquant sur le bouton « Exporter ».
