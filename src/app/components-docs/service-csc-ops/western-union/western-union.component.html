Guide Western Union

Western Union

Pour accéder à Western Union, l’opérateur procède de la manière suivante :

  • Se connecter sur Common Interface ;
  • Cliquer sur Transfert d’argent ;
  • Cliquer sur Western Union ;

En cliquant sur Western Union, la page suivante s’affiche :

1. ENVOI

Western Union comporte deux services de livraison suivant:

  1. Argent minute
  2. Direct to Bank.

1.1. ARGENT MINUTE

L’opérateur procède à l’envoi de la manière suivante : Cliquer sur « Envoi »

Après avoir cliqué sur « Envoi », la fenêtre suivante s’affiche :

  • Sélectionner le type de nom selon que le client possède deux ou encore trois noms
  • Remplir toutes les données de l’expéditeur

N.B :

  1. Si le client qui fait l’envoi est un client qui a déjà passé une opération d’envoi sur la nouvelle plate forme (H2H) sur Common Interface, l’opérateur à la possibilité de le rechercher en mentionnant son numéro de téléphone ou par son nom et puis cliquer sur l’icône « client » juste en dessous du prenom pour le retrouver ; si non l’opérateur rempli toutes les informations du client. Et puis vous cliquez sur le nom afin que les informations s’affichent sur les différents champs et fermez la fenêtre en cliquant sur la croix jaune au dessus à droite.

  2. l’opérateur coche la case « La pièce d’identité a une date d’expiration » pour mentionner la date d’expiration de la carte ; si non il le décoche.

  3. Renseigner le numéro sans débuter avec le préfixe (ex : 890006150, 810000000, …)
    • Renseigner les informations liées au transfert :
      • Sélectionner la devise d’origine
      • Sélectionner le pays de destination
      • Sélectionner la devise de destination
      • Mentionner le montant de la transaction
      • Sélectionner le mode de paiement « ARGENT EN MINUTES »
      • Puis cliquer sur le bouton « Calculer » pour avoir l’estimation
  • Remplir tous les données du bénéficiaire
  • Cocher la case « J’atteste avoir vérifié que toutes les données sont correctes »
  • Cliquer sur le bouton « Valider » pour valider l’opération

Après avoir cliqué sur le bouton « Valider », la fenêtre suivante s’affiche avec le récapitulatif de la transaction :

  • Cliquer sur le bouton « Oui » pour confirmer la transaction

Après avoir confirmé la transaction, le reçu d’envoi s’affiche en deux exemplaires pour imprimer :

L’opérateur a la possibilité de permettre au client (expéditeur) de savoir combien payé pour que bénéficiaire puisse recevoir le montant voulu.

L’opérateur pour ce faire procède de la manière suivante :

  • Cocher la case « calculer par le montant à recevoir »
  • Renseigner le montant que le bénéficiaire doit recevoir
  • Sélectionner le mode de paiement « ARGENT EN MINIUTES »
  • Cliquer sur le bouton « calculer » pour avoir l’estimation

Après avoir cliqué sur le bouton « calculer », le montant à envoyer par l’expéditeur ainsi que le montant total à remettre à l’opérateur pour valider la transaction seront affichés

1.2. DIRECT TO BANK

La procédure d’envoi Western Union en mode de paiement « Direct to Bank » est la même que l’envoi argent minute. La petite différence se situe au niveau du mode de paiement et les informations concernant ce mode de paiement avec ces informations à remplir en fonction du pays de destination.

2. RETRAIT

L’opérateur procède au retrait de la manière suivante :

  • Cliquer sur « Retrait »

Après avoir cliqué sur « Retrait », la fenêtre suivante s’affiche :

  • Insérer le MTCN qui est un code à 10 chiffres
  • Sélectionner la devise de paiement
  • Cliquer sur le bouton « rechercher »,

Après avoir cliqué sur le bouton « rechercher », la page suivante s’affiche avec les informations liées aux détails du paiement ainsi les noms de l’expéditeur et du bénéficiaire :

  • Renseigner les informations obligatoires du bénéficiaire

N.B : l’opérateur coche la case « La pièce d’identité a une date d’expiration » pour mentionner la date d’expiration de la carte ; si non il décoche.

Après avoir cliqué sur le bouton « Valider », la fenêtre suivante s’affiche avec le récapitulatif de la transaction :

Après avoir confirmé la transaction en cliquant sur le bouton « OUI », le reçu de retrait s’affiche en deux exemplaires pour imprimer

3. AUTRES SERVICES

Cette rubrique ne comporte que le statut de transfert qui permet de vérifier si le MTCN est disponible ou est déjà payé.

L’opérateur procède de la manière suivante :

  • Cliquer sur « Autres services »,
  • Cliquer sur « Statut du Transfer »

L’opérateur procède de la manière suivante :

  • Renseigner le MTCN
  • Cliquer sur le bouton « Recherche »

Après avoir cliqué sur le bouton « recherche », le statut de la transaction est mentionné :

4. TRANSACTIONS

Cette dernière rubrique est le Reporting qui reprend toutes les transactions Western Union passées dans Common Interface dans l’agence

L’opérateur a accès au Reporting de ces transactions en procédant de la manière suivante :

  • Cliquer sur « Transactions »
  • Choisir le type de transaction Envoi ou Retrait
  • Renseigner un MTCN particulier pour rechercher une transaction bien déterminée, si non passé outre.
  • Sélectionner la « Date début et Date de fin » afin d’obtenir les transactions d’une période bien déterminée.
  • Sélectionner la devise ou pas
  • Cliquer sur le bouton « Filtrer » afin d’afficher la (ou les) transaction(s).
