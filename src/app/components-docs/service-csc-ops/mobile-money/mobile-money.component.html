Pour y accéder, voici les étapes à suivre :

  • VAS

Ensuite cliquer sur :

  • TELCO

1. Mobile Money

Pour procéder à l’opération de Recharge Unités, il faut cliquer sur l'onglet Mobile money

  • Selectionner le type de transaction et type de compte
  • Entrer un numéro de téléphone ou un numéro marchand
  • Selectionner le devise
  • Entrer le montant
  • Cliquer sur le bouton « Valider »
  • Ensuite confirmer l’opération en cliquant sur le bouton Oui

4. Reporting

L’opérateur sait voir ses opérations des transactions en cliquant sur l'onglet « Transactions » puis sur Mobile money

Après avoir cliqué sur une des options de l'onglet Transaction la page suivante s’affiche

L'operateur peut filtrer les transactions par

  • Date
  • Devise

Exporter

Pour exporter la transaction, sur la page suivante

  • Cliquer sur le bouton Exporter,
  • insérez le nom du fichier,
  • puis cliquer sur Ok
