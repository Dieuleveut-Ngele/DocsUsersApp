Guide Sababalar Opérateur

SABABALAR OPERATEUR

Pour se connecter il faut suivre les étapes suivantes :

  • Saisir le lien suivant sur la barre d’adresse du navigateur : https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Aller dans le menu VAS

Cliquer sur « SABABALAR »

Et la page d’accueil de SABABALAR apparaitra

Sur la page d'accueil Sababalar outre l'accueil il y a deux onglets :

  • Création ticket : permet de créer les tickets saba balar
  • Reporting tickets : permet de visualiser les rapports des tickets crées

1. CREATION TICKETS

Pour créer un ticket :

  • Cliquez sur l’onglet "Création ticket",
  • Choisissez le jeu : Sababalar ou Numéro Million
  • Et cliquer sur "Jouer"

1.1. Sababalar

Si vous choisissez de jouer à Sababalar, vous accéderez à la page suivante :

  • Inserer le numéro de téléphone du client
  • Choisir 6 numéros ou soit vous pouvez les choisir d’une manière aléatoire en cliquant sur le bouton "Aléatoire"
  • Ensuite cliquez sur "Valider" (et recommencer si vous voulez jouer plusieurs numéros gagnant)
  • Une fois que tes choix sont chargés, Insérer un code campagne si il y en a, puis cliquer sur le bouton "Payer"
  • Pour confirmer le paiement du ticket, cliquer sur "Oui" pour confirmer et "Non" pour annuler et recommencer
  • Un reçu apparaîtra, l’imprimer en cliquant sur "IMPRIMER"

1.2. Numéro Million

Si vous choisissez de jouer au Numéro Million, vous accéderez à la page suivante :

  • Renseigner le numéro téléphone du client
  • Code Campagne (Facultatif)
  • Mettre Le nombre des paris (Max 10)
  • En suite cliquer sur "Payer"
  • Pour confirmer le paiement du ticket, cliquer sur "Oui" pour confirmer et "Non" pour annuler et recommencer
  • Un reçu apparaîtra, l’imprimer en cliquant sur "IMPRIMER"

2. REPORTING TICKETS

Cliquez sur le menu "Reporting Tickets" pour afficher le rapport de création des tickets. Vous pouvez filtrer les résultats en fonction des critères suivants :

  • Période (Date de début et Date de fin),
  • Identifiant du ticket (Ticket ID),
  • Numéro de téléphone,
  • Identifiant du point de vente (Pos ID),
  • Statut.

Après avoir appliqué les filtres et affiché le rapport, vous pouvez l'exporter au format Excel en cliquant sur le bouton "Exporter".
