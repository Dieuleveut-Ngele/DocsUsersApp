Guide Bleusat

Bleusat

Pour se connecter, l’opérateur saisit le lien ci-dessous :

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrez votre identifiant et mot de passe, puis validez.

Pour accéder, cliquez sur :

  • VAS

Ensuite, cliquez sur :

  • Bleusat

1. Abonnement

Une fois que l’opérateur accède à Bleusat, la page suivante s’affiche :

Pour passer les opérations Bleusat, l’opérateur procède de la manière suivante :

  • Cliquer sur l'onglet « Abonnement » Sélectionner la devise (USD ou CDF)
  • Cliquer sur le bouton « Charger Bouquets »
  • Sélectionner le bouquet
  • Enter le numéro de la carte Bleusat (16 chiffres)
  • Saisir le numéro de téléphone du client en débutant par 243 (…)
  • Cocher la case « J'atteste avoir vérifié que toutes les données sont correctes ».

Après validation de l’opération, le reçu suivant s’affiche :

N.B : Un SMS sera envoyé au numéro indiqué dans le formulaire.

2. Reporting

Pour voir les differentes opérations Bleusat, l’opérateur procède de la manière suivante :

  • Cliquer sur l'onglet « Transactions » pour voir la page ci-dessus
  • Filtrer les opérations par rapport à une période donnée (date)
  • Filtrer les opérations par rapport à la devise

L'opérateur peut exporter la liste des opérations Bleusat en cliquant sur le bouton Exporter
