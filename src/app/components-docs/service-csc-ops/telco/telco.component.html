Guide Operateur Telco

Operateur telco

Pour y accéder l’operateur saisi le lien ci-dessous

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, l'opérateur se connecte en entrant son identifiant et son mot de passe, puis valider.

Une fois connecté, la page suivante s’affiche

Pour y accéder, voici les étapes à suivre :

  • VAS

Ensuite cliquer sur :

  • TELCO

Une fois que l’opérateur accède à TELCO, la page suivante s’affiche :

1. Recharge Unités

Pour procéder à l’opération de Recharge unités, il faut cliquer sur l'onglet Recharge unités

  • Entrer le numéro de téléphone
  • Entrer le montant du crédit à envoyer (en USD)
  • Cliquer sur le bouton « Valider »
  • Confirmer l’opération en cliquant sur le bouton « Oui »

2. Recharge Data

Pour procéder à l’opération de Recharge Data, il faut cliquer sur l'onglet Recharge mega

  • Saisir le numéro de téléphone du beneficiaire
  • Sélectionner la devise du payement « CDF, USD ou FCN »
  • Cliquer sur le bouton « Charger Forfaits »
  • Ensuite cliquer sur Menu pour selectionner la validité de votre forfaits « exemple 1 jour »
  • Ensuite cliquer sur Forfait pour sélectionner le forfait de votre choix « exemple : 1000 MB »
  • Une fois le forfait sélectionner le montant de la transaction sera affiché

Par la suite l’opérateur clique sur le bouton Valider, pour une dernière vérification de l’opération Une boite de dialogue est affichée avec les informations de la transaction :

  • Le numéro de téléphone
  • Le forfait choisi et le montant à payer

Pour finir, l’opérateur clique sur le bouton Oui et pour annuler l’opération, l’opérateur clique sur le bouton Non

Après avoir cliquer sur le bouton OUI, un Succès vous sera affiché pour signifier que votre opération a réussie

3. Reporting

L’opérateur sait voir ses opérations des transactions en cliquant sur l'onglet « Transactions » puis sur Recharge unités ou soit Recharge mega

Après avoir cliqué sur une des options de l'onglet Transaction la page suivante s’affiche

L'operateur peut filtrer les transactions par

  • Date
  • Devise

Exporter

Pour exporter la transaction, sur la page suivante

  • Cliquer sur le bouton Exporter,
  • insérez le nom du fichier,
  • puis cliquer sur Ok
