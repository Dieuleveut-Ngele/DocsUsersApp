Guide MoneyGram

MONEYGRAM

Pour se connecter, l’operateur saisi le lien ci-dessous

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Veuillez cliquer sur :

  • MONEYGRAM

Sur la page d'accueil MoneyGram outre l'accueil il y a trois onglets :

  • Envoi : permet d’effectuer les envois
  • Retrait : permet d’effectuer les retraits
  • Transactions : permet de retracer le rapport des envois etdes retraits

1. ENVOI

Pour faire un envoi :

  • Cliquez sur l’onglet « Envoi », puis un formulaire apparaitra

Remplissez le formulaire en sélectionnant :

  • le pays de destination,
  • la devise de destination,
  • le montant à envoyé.
  • Cliquez sur « charger mode paiement »
  • Choisissez le « Mode de paiement » dans le menu déroulant pour afficher le frais et le taxe du transfert, pour savoir le montant total que l’expéditeur doit donner pour ce transfert.
  • Renseignez les informations obligatoires de l’expéditeur et du bénéficiaire,
  • Cochez la case « J'atteste avoir vérifié que toutes les données sont correctes »,
  • Et validez la transaction.

Le récapitulatif apparaîtra, puis cliquez sur « OUI ». Après confirmation de la transaction, un reçu apparaîtra.

2. RETRAIT

Pour faire un retrait :

  • Cliquez sur l’onglet « Retrait », puis un formulaire apparaitra
  • Entrez le code MGRN dans le champ à gauche et cliquez sur le bouton « Rechercher »
  • Le résultat de la recherche apparaîtra dans le champ « INFORMATION DU PAIEMENT »
  • Renseignez les informations obligatoires du bénéficiaire
  • Cochez la case « J'atteste avoir vérifié que toutes les données sont correctes »
  • Et validez la transaction
  • Le récapitulatif apparaîtra, puis cliquez sur « Oui »

Après confirmation de la transaction, un reçu apparaîtra.

3. TRANSACTIONS

L’onglet « Transactions » donne la possibilité de voir le rapport de toutes les transactions effectuées pour une période précise.

Pour vérifier une transaction :

  • Cliquez sur l’onglet « Transactions »
  • Sélectionnez « envois ou retraits » selon le besoin
  • Dans le formulaire, précisez la période en renseignant sur la date Début et date Fin
  • Cliquer sur « Filtrer »

Le bouton d’action « Imprimant » permet la réimpression du reçu.

NB : Si on souhaite :

  • voir les transactions réalisées seulement avec une monnaie quelconque, là il faudrait cliquer sur l’onglet Devise puis sélectionner la monnaie.
  • Voir le status d’une transaction exacte, il faut insérer le numero de la transaction MGRN.
