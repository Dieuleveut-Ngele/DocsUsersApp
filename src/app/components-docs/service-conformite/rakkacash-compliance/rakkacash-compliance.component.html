RAKKACASH COMPLIANCE

Pour se connecter, suivez les étapes suivantes : saisissez le lien commun dans la barre d’adresse du navigateur pour accéder à la page de connexion.

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrez votre identifiant et mot de passe, puis validez.

Dans le menu Transfert d’argent, cliquez sur « RAKKACASH » et la page d’accueil de RAKKACASH apparaîtra.

La page se compose de 4 menus :

  • Reporting Cashin : Permet de visualiser la liste des cashin
  • Reporting Cashout : Permet de visualiser la liste des cashout
  • Identité clients : Permet de visualiser les pièces d’identité des clients
  • Blacklist : Permet de bloquer et débloquer des clients blacklistés

1. REPORTING CASHIN

Sur la page d’accueil, cliquez sur le menu Reporting Cashin

Dans cette page, vous pouvez filtrer les transactions par :

  • Pays
  • Région
  • Ville
  • Agence
  • Code retrait
  • Date début & Date fin
  • Devise
  • Statut
  • Bénéficiaire
  • Déposant
  • Numéro
  • Numéro pièce d’identité
  • ID Transaction

En cliquant sur le bouton Filtrer

Le reporting cashin filtré s’affiche, vous pouvez l'exporter en Excel en cliquant sur le bouton Exporter

2. REPORTING CASHOUT

Sur la page d’accueil, cliquez sur le menu Reporting Cashout

Dans cette page, vous pouvez filtrer les transactions par :

  • Pays
  • Région
  • Ville
  • Agence
  • Code retrait
  • Date début & Date fin
  • Devise
  • Statut
  • Bénéficiaire
  • Déposant
  • Numéro
  • Numéro pièce d’identité
  • ID Transaction

En cliquant sur le bouton Filtrer

Le reporting cashout filtré s’affiche, vous pouvez l'exporter en Excel en cliquant sur le bouton Exporter

3. IDENTITÉ CLIENTS

Sur la page d’accueil, cliquez sur le menu Identité clients

Dans cette page ci-dessus, vous pouvez filtrer les informations clients par :

  • Prénom
  • Nom
  • Postnom
  • Numéro d'identité
  • Type de pièce d’identité
  • PIN

En cliquant sur le bouton Filtrer

4. BLACKLIST

Sur la page d’accueil, cliquez sur le menu Blacklist

Dans cette page ci-dessus vous pouvez :

  • Rechercher un client par son numéro d'identité et au moins un nom en cliquant sur le bouton Rechercher
  • Ajouter un client à la liste des clients blacklistés
  • Exporter la liste des clients blacklistés en Excel en cliquant sur le bouton Exporter

NB : le bouton Supprimer dans la colonne Action permet de débloquer un client blacklisté.
